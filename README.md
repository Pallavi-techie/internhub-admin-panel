
# Admin Panel – InternHub Management System

This is a full-stack **Admin Panel** built to manage internships, jobs, users, content, and internal operations.  
The admin dashboard provides a centralized interface for handling postings, applications, verifications, blogs, and team management.

---

## 🧠 Admin Dashboard Overview

The Admin Dashboard acts as the control center of the platform, allowing administrators to manage both **career-related workflows** and **website content** efficiently.

---

## 🚀 Core Features (Based on Actual Dashboard)

### 📝 Internship & Job Management
- **Post Internship**  
  Create and publish new internship listings with role details.
- **Post Job**  
  Add full-time job openings to the platform.
- **Uploaded Internships**  
  View, manage, and update posted internships.
- **Uploaded Jobs**  
  Manage all published job listings.

---

### 📄 Applications Handling
- **Internship Applications**  
  View and manage incoming internship applications.
- **Job Applications**  
  Review candidates who applied for jobs.

---

### 🛡 Verification System
- **Manage Verifications**  
  Add and manage employee & intern verification records.
- **Verify Credential Page Support**  
  Backend support for credential verification functionality.

---

### 📅 Interview & Scheduling
- **Interview Scheduler**  
  Schedule and manage interviews for applicants.

---

### 👥 User & Team Management
- **Manage Users**  
  Add, update, or manage admin and employee accounts.
- **Manage Team**  
  Add or remove internal team members.

---

### 💬 Communication & Requests
- **User Queries**  
  Handle customer or user inquiries.
- **Connection Requests**  
  Review and respond to connection requests from users or partners.

---

### 📰 Content Management
- **Blog Management**  
  Create and manage blog posts.
- **Create Blog (Quick Action)**  
  Fast access to writing and publishing blogs.
- **View Blog (Quick Action)**  
  Preview published blogs.

---

### 🖼 Slider & Media Management
- **Manage Slider Ads**  
  Upload, reorder, and manage homepage slider advertisements.
- **Manage Clients**  
  Add, edit, and reorder client logos.

---

### ⚡ Quick Actions
- View Blog
- Create Blog
- Job Listings
- Internship Listings

These shortcuts improve admin productivity and workflow efficiency.

---

## 🧩 Frontend Architecture

- Reusable dashboard cards
- Shared navbar across all pages
- Template-based list pages
- Tailwind CSS for consistent UI
- Redux for scalable state management
- Framer Motion for smooth interactions

---

## 🛠 Backend Architecture

- Express-based REST APIs
- Modular routes for each admin feature
- Middleware for authentication & protection
- MongoDB models for structured data handling
- Environment-based configuration

---

## 🎯 Purpose of This Project

This Admin Panel was developed to:
- Simulate a real-world SaaS admin system
- Manage recruitment and content workflows
- Practice role-based backend design
- Build an industry-level dashboard application
- Prepare for internships and professional full-stack roles
